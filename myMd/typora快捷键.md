# typora

## 1.快捷键

无序列表：“-”+space
有序列表：数字+“.”+space
任务列表：“-” + space + 空格 文字
标题：ctrl + 数字
表格：Ctrl + t
生成目录：[TOC]+回车
选中一整行：ctrl + l
选中单词：ctrl + d
选中相同格式的文字：ctrl + e
跳转到文章开头：ctrl + home
跳转到文章结尾：ctrl + end
搜索：ctrl + f
替换：ctrl + h
引用： “>” + space
代码块：ctrl + shift+K
加粗：ctrl + b
倾斜：ctrl + i
下划线：ctrl + u
删除线：ctrl + shift + s
插入图片：ctrl + shift + i
插入链接：ctrl + k

## 2.符号指令(英文符号)

tab 上边的`输入三个 再写个语言，就是改语言的代码框

`print（）`    //两个该符号包裹起来为代码格式
